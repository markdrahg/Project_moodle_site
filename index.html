Overview

Welcome to the documentation for the Moodle Job Plugin. This plugin is currently in its alpha stage and is developed using PHP. It was initiated by Mark Drah, the founder of VisionMark.

The plugin's code is open source, encouraging developers to contribute, make improvements, or create personal modifications. The primary goal is to expand its accessibility and functionality across the Moodle environment, fostering collaboration and innovation in the community.

Installation

Follow these steps to install and configure the Moodle Job Plugin:

Step 1: Download the plugin's zip file here.

Install the downloaded zip file through the Moodle plugin installation panel.

Step 2: After a successful installation, navigate to the following URL on your Moodle page:

http://localhost/moodle/local/jobapi/

Copy this URL and set it to a URL activity. This will serve as the job listings page.

Step 3: Navigate to the admin job control panel using this URL:

.../local/jobapi/Jobs_And_Applicants_Control.php

Copy this URL and set it to a URL activity. This will serve as the admin job control panel.

Usage

For Users

  • View available job listings, including open positions and their deadline dates.
  • Apply for a job by submitting your resume.
  • Each user can apply to a job only once.

For Administrators

  • Add, update, and delete job listings.
  • Download individual or bulk resumes for specific job applications.
  • Generate and download an Excel sheet of applicants for any job.
  • Utilize the advanced search feature to query and manage applicant details efficiently.
